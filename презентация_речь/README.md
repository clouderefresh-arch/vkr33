# Презентация и речь к ВКР Кострова

Материалы к защите выпускной квалификационной работы на тему
**«Применение интернет-технологий в коммерческой деятельности предприятий»**
(студент — Костров; научный руководитель — к.э.н., доцент Саенко М. Ю.).

Презентация и доклад подготовлены **по структуре шаблона** `шаблон-СПО.pptx`
и **по структуре шаблона речи** `речь.doc`, заполнены **данными из ВКР**
(`ВКР_Костров_20260521_1832.docx`).

## Структура презентации (10 слайдов)

Структура шаблона сохранена; текст-заполнители заменены данными ВКР,
демонстрационный брендинг примера («Гемотест») удалён.

1. **Титульный** — тема, студент, руководитель.
2. **Актуальность, объект, предмет, цель, задачи.**
3. **База исследования** — ОЭБиПК УВД по ЮЗАО и модельная группа П-1 — П-5.
4. **Ключевые цифры** — рынок э-торговли РФ (9,1 трлн ₽, +35 %), доля онлайн 15–16 %,
   5 предприятий, диапазон ИЦД 0,43–0,86, KEO ≈ 9,5 (цветные карточки шаблона).
5. **Анализ цифровых каналов модельной группы** (глава 2) — таблица по П-1 — П-5,
   коэффициент цифровой интенсивности Kди.
6. **Защищённость каналов и индекс цифрового доверия (ИЦД)** — уровень мер защиты и ИЦД.
7. **Ключевые уязвимости и угрозы экономической безопасности.**
8. **Концептуальные предложения** — программа из 4 звеньев (профилактика, аналитика,
   цифровой мониторинг, межведомственный обмен), 7 мероприятий.
9. **Экономическая эффективность** — динамика показателей «до — после», KEO = 9,5,
   NPV ≈ 9,25 млн ₽, окупаемость 5,3 мес.
10. **Спасибо за внимание.**

## Структура речи

Соответствует шаблону `речь.doc`: приветствие ГАК → тема и актуальность →
объект → предмет → цель и задачи → база исследования → результаты по главам →
практическая значимость и научная новизна → вывод → «Спасибо за внимание».
В тексте курсивом проставлены пометки `[Слайд N]` для синхронизации с презентацией.

## Как пересобрать материалы

Требуются Python-пакеты `python-docx`, `python-pptx`; для экспорта в PDF — LibreOffice.

```bash
cd презентация_речь
python3 build_presentation.py   # -> Презентация_Костров_ВКР.pptx
python3 build_speech.py         # -> Речь_Костров_ВКР.docx

# Экспорт в PDF (опционально)
soffice --headless --convert-to pdf --outdir . "Презентация_Костров_ВКР.pptx"
soffice --headless --convert-to pdf --outdir . "Речь_Костров_ВКР.docx"
```

## Примечания

- Все числовые данные (выручка, доли онлайн-канала, Kди, ИЦД, KEO, NPV, прогнозы
  «до — после») взяты из таблиц и текста ВКР.
- Названия предприятий обезличены (П-1 — П-5) в соответствии со ст. 161 УПК РФ и
  ФЗ № 152-ФЗ — так же, как в самой работе.
